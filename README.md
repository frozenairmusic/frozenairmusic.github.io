# Frozen Air

Electronic music project website built with Next.js and deployed on GitHub Pages.

## Development

```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Deployment

Automatically deploys to GitHub Pages via GitHub Actions when pushing to master branch.

See [DEPLOYMENT.md](DEPLOYMENT.md) for detailed deployment instructions.

## Features

- Scroll-based animations
- GDPR-compliant cookie consent
- Google Analytics integration
- Responsive design
- Custom domain: frozenairmusic.com
